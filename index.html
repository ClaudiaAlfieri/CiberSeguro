Tipos de ameaças

Phishing

Emails ou mensagens falsas que imitam empresas reais para roubar os teus dados.

Passwords fracas

Usar '123456' ou o teu nome como password é um convite para os hackers.

Malware

Programas maliciosos que se instalam no teu dispositivo sem o teu conhecimento.

Wi-Fi público

Ligar-te a redes abertas pode expor os teus dados a desconhecidos na mesma rede.
