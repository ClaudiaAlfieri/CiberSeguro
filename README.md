## 📌 About the Project

**CiberSeguro** is a front-end website focused on everyday digital security, developed as a practical project for the  **CET TPSI** course at **ATEC — Training Academy**.

The goal is to present real digital risks, best practices, and interactive content in a clear and accessible way — demonstrating autonomy, organization, and creativity in applying web development concepts.🎯

---

## 🌐 Website Sections

| Section | Description |
|---|---|
| 🏠 Home | Hero section with site introduction and author’s name |
| ⚠️ Types of Threats | Clickable cards with the main digital threats — opens a modal with more detailss |
| 🧠 Quiz | Interactive 10-question quiz to test digital security knowledge |
| 💡 Security Tips | Tips organized in tabs: Passwords, Browsing, and Devices |
| 📬 Get Tips by Email | Subscription form with validation and confirmation message |

---

## ⚠️ Covered Threats

- 📧 **Phishing** — fake emails and messages that steal your data
- 🔑 **Weak Passwords** — why "123456" is an open door for hackers
- 🛡️ **Malware** — viruses, ransomware, and spyware
- 🌐 **Public Wi-Fi** — the risks of open networks

---

## 🛠️ Technologies and Concepts Applied

- **HTML5** — semantic structure and content organization
- **CSS3**
  - Flexbox and Grid layout
  - Media queries for responsiveness (desktop, tablet, and mobile)
- **JavaScript**
  - DOM manipulation
  - Modal with dynamic open/close behavior
  - Interactive quiz using switch and conditions
  - Tabs with dynamic content switching
  - Form with validation and user feedback
  - onclick and onchange events
